# python-flask
This project is for learning flask architecture and having basic grasp of python framework

#setting up venv on windows:
>pip install virtualenv
>virtualenv venv
>venv\scripts\activate

#installing flask:
>pip install Flask

#running application:
>set FLASK_APP=App.py
>flask run
